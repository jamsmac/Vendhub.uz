# VendHub — Спецификация будущего клиентского сайта

**Тип:** Информационный сайт (НЕ личный кабинет)
**Назначение:** Показать продукты, автоматы, акции и бонусную программу. Привлечь клиентов в мобильное приложение
**Источник данных:** API из VHM24 (динамические данные, управление через админ-панель)
**Дизайн-система:** "Warm Brew" (кремовые тона, кофейная палитра)

## HEADER — Шапка сайта

### Расположение
Фиксированный (`fixed top-0 w-full z-50`)

### Внешний вид
- **Без скролла:** `bg-espresso-dark` (тёмно-коричневый, сплошной)
- **При скролле:** `bg-espresso-dark/95 backdrop-blur-lg shadow-xl` (полупрозрачный с размытием)

### Содержимое (слева направо)

```
┌────────────────────────────────────────────────────────────┐
│ [☕ VendHub]   Главная  Автоматы  Меню  Выгода             │
│  Coffee&Snacks Партнёрство  О нас               [Войти →] │
└────────────────────────────────────────────────────────────┘
```

#### Логотип (слева)
- Иконка: `vendhub-logo.png` (40×40)
- Текст: **"VendHub"** (белый, DM Sans bold)
- Подтекст: "Coffee & Snacks" (cream/60, мелкий)

#### Навигация (центр, desktop — 6 пунктов)
| # | Пункт | Секция ID | Поведение |
|---|-------|-----------|-----------|
| 1 | Главная | `home` | Scroll to top |
| 2 | Автоматы | `map` | Scroll to section |
| 3 | Меню | `menu` | Scroll to section |
| 4 | Выгода | `benefits` | Scroll to section |
| 5 | Партнёрство | `partner` | Scroll to section |
| 6 | О нас | `about` | Scroll to section |

- **Стиль активного:** `bg-white/10 text-caramel rounded-lg px-3 py-1.5`
- **Стиль обычного:** `text-white/60 hover:text-white transition`
- **Скрыто на мобильном** (`hidden lg:flex`)

#### Кнопка «Войти» (справа)
- Стиль: `btn-caramel` (gradient caramel → caramel-dark, белый текст)
- Текст: **«Войти»**
- Иконка: стрелка вправо (→)
- Действие: переход на мобильный ЛК (Telegram Mini App)
- Пока приложение не готово: placeholder URL / toast «Скоро»

#### Мобильное бургер-меню (≤lg)
- Иконка: ☰ (3 полоски, белые)
- При нажатии: выпадающий dropdown под header
- Анимация: fadeIn
- Содержит: 6 пунктов навигации + кнопка «Войти» (полная ширина)
- При скролле / выборе пункта: закрывается
